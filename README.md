# The-13-Hour-Wall-Troubleshooting-the-IRS2092-500W-Amplifier
Building your own high-power audio gear is a rewarding experience—until it isn't. In this project, I spent 13 hours deep in the guts of an IRS2092S-based 500W Class D Amplifier, only to come face-to-face with the reality that sometimes, despite your best efforts, the silicon wins.
